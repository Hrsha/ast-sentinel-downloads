# AST-Sentinel GCS — Downloads

**Public download mirror.** Compiled, signed, notarized macOS binaries of
AST-Sentinel — a professional BVLOS-grade Ground Control Station for
MAVLink-compatible drones (ArduPilot / PX4).

Source code is proprietary and lives in a private repository.

## Latest release

See [Releases](../../releases) for the most recent build.

### macOS
- **Apple Silicon (M1/M2/M3/M4)**: `AST-Sentinel-X.Y.Z-arm64.dmg`
- **Intel Macs**: `AST-Sentinel-X.Y.Z-universal.dmg` (works on both architectures)

All builds are signed with the Astroc Developer ID and notarized by Apple.
You should not see a Gatekeeper warning on first launch.

## Install
1. Download the DMG matching your Mac.
2. Open the DMG (double-click).
3. Drag `AST-Sentinel.app` to `/Applications`.
4. Launch from `/Applications`.
